# Shkoba Multiplayer Game

A real-time multiplayer implementation of **Shkoba**, a popular card game, using **Flutter** for the client and **Socket.IO** for the server.  

Features:
- Play Shkoba with friends online.
- Capture cards based on matching or summing table cards.
- Track Shkobba counts and captured cards.
- Real-time updates using Socket.IO.
- Responsive and animated card UI.
